UKRmol+ Tools
=============

Jakub Benda (c) 2025


### Information

Helper scripts and programs based on UKRmol+.

 - polarizability.f90: Evaluates the static or dynamic dipole polarizability of a molecule
   given cdenprop outputs.
 - build-ukrmolp.sh: Builds a shell installer of UKRmol+ with all dependencies.

### Building

Customize the Makefile and build by `make`.
